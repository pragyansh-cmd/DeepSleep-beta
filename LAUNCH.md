# Launch Plan

You cannot guarantee 100k stars, but you can maximize the odds that people instantly understand the product.

## Positioning

Lead with:

> The open-source background agent for local models.

Follow with:

- zero-cost
- idle-time dreaming
- 3-layer memory
- `ds init`, `ds`, `ds dream`

## Demo sequence

Use a 45-60 second terminal demo:

1. `ds init`
2. open or edit two files
3. run `ds dream --once`
4. show `.deepsleep/memory.json`
5. ask `What was I doing?`
6. run `ds doctor`

## Launch assets to prepare

- short GIF of the terminal flow
- one screenshot of the interactive prompt
- pinned tweet/post with a 30-second story
- GitHub topics: `local-ai`, `ollama`, `developer-tools`, `agent`, `cli`, `memory`
- a clean repo banner and social preview image

## Message angles to test

- "Claude-style memory, but local and open source"
- "Your repo dreams while you sleep"
- "A zero-cost coding agent powered by Ollama"
- "Ask your terminal: what was I doing?"

## First week priorities

- merge simple PRs quickly
- clip one short demo every day
- reply fast to install issues
- add obvious requested integrations
- keep the README above-the-fold story extremely clear
